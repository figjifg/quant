# Audit-first Framework

Status: system default.

## Core Rule

Every new family starts with A0 Audit Gate before alpha interpretation.

No family result is interpreted as alpha before the audit gate is passed.

## A0 Audit Gate

1. data lineage
2. PIT availability
3. survivorship safety
4. corporate action handling
5. calendar / tradability
6. daily NAV
7. no leverage unless explicit
8. random/date-matched control
9. concentration / top contributor
10. cost / tax / turnover

## Operating Principle

Audit-first is the default research and engineering framework. Strong headline
results are treated as untrusted until the audit gate verifies the data,
timing, execution, controls, and accounting.
