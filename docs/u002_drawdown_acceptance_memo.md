# U002 Drawdown Acceptance Memo

Status: pre-commitment memo for `P08_IEF30` live pilot behavior.

## 목적

`P08_IEF30`는 drawdown을 제거하는 전략이 아니다. 이 문서는 손실 구간별
허용 행동과 금지 행동을 사전에 고정해 live pilot 중 임의 판단을 막는다.

## Drawdown 구간

| 구간 | NAV drawdown | 해석 |
|---|---:|---|
| 정상 변동성 | -5% | 일상적 변동. 조치 없음. |
| 정상 변동성 | -10% | 불편하지만 사전 허용 범위. |
| 정상 변동성 | -15% | 리밸런싱 외 행동 금지. |
| 고통 구간 | -20% | pilot 지속 가능성 점검, 전략 변경 금지. |
| 고통 구간 | -25% | 운영 오류와 paper/live 괴리만 점검. |
| 위기 구간 | -30% | 사전 폐기 조건 충족 여부 확인. |
| 위기 구간 | -35% | live pilot 축소/중단 조건 검토. |

## 금지 행동

- 임의 전량 매도 금지.
- QQQ만 제거 금지.
- IEF만 추가 금지.
- 뉴스, 전망, 금리 코멘트만 보고 리밸런싱 변경 금지.
- H001 sleeve 제거 금지.
- shadow 후보의 최근 수익률만 보고 live 교체 금지.

## 허용 행동

- 정해진 분기 리밸런싱 실행.
- N002-B/N001-B shadow 결과 비교.
- 세금 손실 처리 검토.
- paper/live NAV 괴리 원인 분석.
- 브로커 체결, 환전, 세금 lot 기록 점검.

## Live Pilot 중단 조건

아래 중 하나라도 발생하면 신규 투입과 증액을 중단한다.

- paper/live NAV 괴리가 누적 2%p를 초과하고 설명되지 않음.
- 데이터 오류로 기준 NAV 또는 목표 비중 산출이 불가능함.
- 세금 lot 추적 실패.
- 시장가 주문, 임의 비중 변경, 무승인 대체 상품 사용 등 규칙 위반.
- 계좌 제약 때문에 목표 sleeve 노출을 더 이상 재현할 수 없음.

중단은 전략 폐기가 아니다. 중단 후에는 원인, 영향 금액, 재개 조건을 별도
운영 로그에 기록한다.
