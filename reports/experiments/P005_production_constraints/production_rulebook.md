# P005 Production Rulebook

- Carrier: D013 unchanged.
- Single-name cap: 25%.
- Top-2 aggregate cap: 50%.
- Liquidity floor: 20-day average traded value >= 5,000,000,000 KRW.
- Status exclusion: exclude rows flagged as trading halt, administrative issue, or delisting when panel flags exist.
- Turnover cap: one-way quarterly turnover <= 100%.
- AUM cap: 10,000,000,000 KRW; defined as a production constraint only and not applied to backtest fills.
