# D013 — D009 carrier + 임계값 -0.2 (새 챔피언 후보)

## 상태
계획됨

## 이게 무슨 ticket 인가

D011 (D009 의 임계값 흔들기 테스트) 에서 임계값을 -0.2 로 낮추면
샤프 0.53, 누적 +254% 라는 결과가 나왔다. D-family 전체에서 가장 좋은
숫자다. D001 의 최고 (샤프 0.48, 누적 +129%) 보다 훨씬 낫다.

하지만 D011 은 임계값을 흔들어본 테스트일 뿐이고, **임계값 -0.2 를
정식 기준 전략으로 사전 등록한 적이 없다**. 사후적으로 "그게 더 좋았네"
라고 채택하면 결과를 보고 정한 것 (data snooping) 이 된다.

D013 = 임계값 -0.2 를 **정식 기준 전략으로 사전 등록**하고, 동일한
검증 절차 (견고함 테스트 D014, D015) 거친다.

## 메커니즘 설명

D009 의 합성 점수 분포: 평균 -0.03, 표준편차 0.42.

- 임계값 0.0 → 점수가 0 위인 분기만 매수. 분포가 0 근처이니
  매수 분기가 23%
- 임계값 -0.2 → 점수가 -0.2 위인 분기 매수. 약간만 비관 아니면
  매수. 매수 분기가 38% 로 늘어남
- D009 의 평균 점수 -0.03 자체가 D001 의 -0.22 보다 훨씬 0 에 가까움
  → D009 는 원래 "평소엔 약간 중립" 인 분포
- 그래서 -0.2 까지 살짝 낮춰서 매수 분기 늘리는 게 적합

D001 에서 임계값 -0.2 로 낮추는 것과는 다른 이야기:
- D001 합성 점수 평균 -0.22 → 임계값 -0.2 면 사실상 절반 가까이 매수,
  D001 의 D007 에서 보면 샤프가 떨어졌음
- D009 합성 점수 평균 -0.03 → 임계값 -0.2 면 적당히 많이 매수,
  D011 에서 샤프가 올라감

즉 임계값은 그 carrier 의 합성 점수 분포에 맞춰야 함.
